# Design System

## Color Palette

| Token | Value | Usage |
|---|---|---|
| Background | `#FAFAF8` | Page background |
| Text primary | `#1A1A18` | Body text, headings |
| Text secondary | `#888880` | Captions, metadata, muted labels |
| Accent | `#8B1A1A` | Logo, section labels, hover states, key headline phrases — use sparingly |
| Border | `#E8E4DC` | Dividers, card borders |
| Card background | `#ffffff` | Cards, panels |
| Footer background | `#F0EDE8` | Footer section |
| Stats background | `#1A1A18` | Dark contrast section for stats/callouts |

## Typography

- **Font:** System sans-serif
- **Headings:** `font-weight: 500`
- **All-caps labels:** `font-size: 11px`, `letter-spacing: 2px` — used for section labels and category tags

## Style

- Clean, minimal, light editorial feel
- Cards: white background, `0.5px` border (`#E8E4DC`), `border-radius: 12px`, no heavy shadows
- Buttons: dark filled (`background: #1A1A18`, white text) or outlined (`border: 1px solid #1A1A18`, transparent background)
- Nav: light background with bottom border, clean and minimal

## Accent Color Usage

The accent (`#8B1A1A`) appears in:
- Logo `.you` wordmark
- Section labels (all-caps, small)
- Link and button hover states
- Key headline phrases or highlighted words
